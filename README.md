# SmartCommunity
环泰智慧社区
简介：  
------
一款智能家居的配套应用，基于Android平台，使用Android Studio开发，集成了机智云，Bomb云,百度AI等平台SDK.功能主要包括智能家居，生活服务，我的物业.正不断完善中....
